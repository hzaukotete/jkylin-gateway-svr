# j-kylin-gateway-svr 

## 一句话描述
基于 gin + go 的http接入层的网关

## Quick Start
```
git clone xxxxxxxx
cd xxxxxxx
go mod tidy
```

## Support
1. 支持签名校验
```
实现难度：简单
需要设计签名方案,接口注册方案
```
2. 支持服务粒度、用户粒度的权限校验
```
实现难度：中等
需要先设计开发基础服务再考虑这个问题
```
3. 支持敏感字段加解密 RSA_OAEP_with_AES_256_GCM
```
实现难度：简单
封装加解密方法即可
```
4. 支持各种格式请求透传转发，支持https转http
```
实现难度：简单
基础功能，需要看下有没有必要在这里做htpps转http
```
5. 支持流水日志和监控功能
```
实现难度：难
需要评估市面上主流的日志组件，监控组件
```
6. 支持service按user灰度发布的能力
```
实现难度：中等
需要先设计开发基础服务再考虑这个问题，最简单的想法就是根据请求头中的用户信息分发路由
```
